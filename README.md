# FishNet-Sample

本リポジトリは「Fish-NetでUnityのマルチプレイゲームを作ろう」のサンプルプロジェクトです．

# 検証環境

 * Unity 2021.2.8f1
 * FishNet 1.1.5
 * ParrelSync 1.5.0
